# SocialSync Calendar

A calendar app for planning and managing social media posts. View by calendar or list, edit posts, sync with Google Sheets, and use AI to generate content.

## Run locally

- **Prerequisites:** Node.js
- Install: `npm install`
- Add `GEMINI_API_KEY` to `.env.local` (optional, for AI features)
- Start: `npm run dev`

## Deploy to GitHub Pages

See the short steps at the top of [.github/workflows/deploy-pages.yml](.github/workflows/deploy-pages.yml). After deployment, the app is at `https://<username>.github.io/<repo-name>/`.
